
# Unlimited Reverse IP



## Installation



```bash
  install python 3.X 
  install modules "requests" "colorama"
  pip install requests colorama
  create a .txt file that contains ips
  run reverse_ip.py 
  Thread : 20 (Recommanded)
  Results will be saved in domains.txt file.

```
    
## Features

- Very Fast
- Live previews
- Saves results
- Cross platform
- Proxyless
